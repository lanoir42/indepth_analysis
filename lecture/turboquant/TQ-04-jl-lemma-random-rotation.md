# TQ-04: JL Lemma와 랜덤 회전의 마법

> **목표**: Johnson-Lindenstrauss Lemma를 직관적으로 이해하고, 랜덤 회전이 양자화를 돕는 메커니즘을 파악
> **선수 지식**: 벡터 노름, 내적, 행렬 곱셈, 정규분포
> **소요 시간**: 40분

---

## 1. 고차원의 직관에 반하는 성질

### 핵심 질문

> "128차원 벡터를 랜덤하게 회전하면, 왜 양자화에 좋아지는가?"

답을 이해하려면 먼저 **고차원 공간의 놀라운 성질**을 알아야 합니다.

### 고차원에서 벌어지는 이상한 일들

**1. 대부분의 부피가 "껍질"에 있다**

$d$차원 단위 구의 부피 중, 중심 근처의 비율:

$$\frac{\text{Vol}(B_d(r))}{\text{Vol}(B_d(1))} = r^d$$

$d = 128$, $r = 0.95$이면: $0.95^{128} \approx 0.001$

> 128차원 단위 구의 부피 중 **99.9%가 바깥쪽 5% 껍질**에 집중!

**2. 랜덤 벡터는 거의 직교한다**

두 랜덤 단위 벡터 $u, v \in \mathbb{R}^{128}$의 내적:

$$\langle u, v \rangle \sim \mathcal{N}\left(0, \frac{1}{128}\right) \approx 0 \pm 0.09$$

> 128차원에서 두 랜덤 벡터는 **거의 확실히 수직**입니다.

**3. 좌표값이 집중된다**

단위 구 위의 균등 분포에서 임의의 좌표 $u_i$:

$$u_i \approx \mathcal{N}\left(0, \frac{1}{d}\right)$$

$d$가 크면 각 좌표는 0 근처에 **좁게** 집중.

> **이 세 성질이 TurboQuant의 수학적 기초입니다.**

## 2. Johnson-Lindenstrauss Lemma (1984)

### 정리 (JL Lemma)

> $n$개 점 $\{x_1, \ldots, x_n\} \subset \mathbb{R}^d$에 대해, $k = O(\varepsilon^{-2} \log n)$ 차원의 **랜덤 선형 사상** $f: \mathbb{R}^d \to \mathbb{R}^k$가 존재하여, 모든 $i, j$에 대해:
>
> $$(1 - \varepsilon)\|x_i - x_j\|^2 \leq \|f(x_i) - f(x_j)\|^2 \leq (1 + \varepsilon)\|x_i - x_j\|^2$$

### 쉬운 말로

"고차원의 점들을 **랜덤 행렬**로 저차원에 사영(projection)해도, **쌍별 거리가 거의 보존**된다."

### 구체적 숫자

$n = 100{,}000$ (토큰 수), $\varepsilon = 0.1$ (10% 오차 허용):

$$k = O\left(\frac{\log 100{,}000}{0.01}\right) = O(1{,}151)$$

128차원 벡터를 ~1,000차원으로 사영해도 거리가 10% 이내로 보존!

(실제로는 원래 차원보다 적은 차원으로 줄이는 데 사용하지만, 핵심 통찰은 **랜덤 투영이 구조를 보존**한다는 것)

## 3. JL이 작동하는 이유: 측도의 집중

### 측도의 집중 (Concentration of Measure)

고차원에서 **랜덤 투영은 놀라울 정도로 안정적**입니다. 이유:

고정된 단위 벡터 $u$를 $k$차원 랜덤 부분공간에 투영하면:

$$\|Ru\|^2 \approx 1 \pm O\left(\sqrt{\frac{1}{k}}\right)$$

여기서 $R$은 랜덤 투영 행렬.

### 직관

동전을 $k$번 던지면, 앞면 비율은 $1/2 \pm O(1/\sqrt{k})$에 집중합니다.
랜덤 투영도 같은 원리: 많은 랜덤 방향으로 사영하면, 각 방향의 기여가 **평균으로 집중**.

### 증명 스케치 (핵심 부등식만)

$R \in \mathbb{R}^{k \times d}$, 각 원소 $R_{ij} \sim \mathcal{N}(0, 1/k)$

고정 단위 벡터 $u$에 대해:

$$P\left[\left|\|Ru\|^2 - 1\right| > \varepsilon\right] \leq 2\exp\left(-\frac{\varepsilon^2 k}{8}\right)$$

$k$가 크면 이 확률이 **지수적으로 0에 수렴**.

모든 $\binom{n}{2}$ 쌍에 Union Bound 적용:

$$k \geq \frac{8}{\varepsilon^2} \ln n \implies \text{모든 쌍이 동시에 보존}$$

## 4. 랜덤 회전이 양자화를 돕는 메커니즘

### TurboQuant에서의 사용

TurboQuant는 JL 투영을 "차원 축소"가 아니라 **"분포 균등화"**에 사용합니다.

**Step 1**: 임의의 벡터 $x \in \mathbb{R}^d$에 **랜덤 직교 행렬** $R$을 곱함:

$$y = Rx$$

### 랜덤 회전 전후의 변화

**회전 전**: $x$의 좌표들은 **불균등한 분포** — 어떤 좌표는 크고 어떤 좌표는 작음

```
x의 좌표 분포 (예시):

   ▓                   ← 큰 좌표
   ▓▓
   ▓▓▓
   ▓▓▓▓
   ▓▓▓▓▓▓
   ▓▓▓▓▓▓▓▓▓▓▓
  ─────────────────
  좌표 값

  → 분포가 불규칙하고 데이터마다 다름
  → 양자화기를 설계하려면 데이터를 알아야 함 (캘리브레이션 필요)
```

**회전 후**: $y = Rx$의 좌표들은 **균등하고 예측 가능한 분포**

```
y의 좌표 분포:

            ▓▓▓
          ▓▓▓▓▓▓▓
        ▓▓▓▓▓▓▓▓▓▓▓
      ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
  ─────────────────────────
           0

  → 항상 N(0, ||x||²/d)에 가까운 벨 모양
  → 원래 데이터와 무관! (Data-Oblivious)
  → 양자화기를 미리 설계할 수 있음
```

### 왜 이것이 혁명적인가?

| | 회전 전 | 회전 후 |
|---|---|---|
| 좌표 분포 | **데이터에 의존** | **항상 동일** (가우시안) |
| 양자화기 설계 | 캘리브레이션 **필요** | 캘리브레이션 **불필요** |
| 좌표 간 상관 | **있을 수 있음** | **거의 독립** |
| 정규화 상수 | **저장 필요** | **불필요** (분포가 알려져 있으므로) |

## 5. Beta 분포와 집중 현상

### 회전 후 좌표의 정확한 분포

단위 구 $S^{d-1}$ 위의 균등 분포에서, 각 좌표 $u_i$의 **제곱**:

$$u_i^2 \sim \text{Beta}\left(\frac{1}{2}, \frac{d-1}{2}\right)$$

### 고차원에서의 근사 (CLT)

$d \to \infty$일 때:

$$u_i \xrightarrow{d} \mathcal{N}\left(0, \frac{1}{d}\right)$$

$d = 128$이면 이 근사는 **매우 정확**합니다.

### 핵심 성질: 좌표 간 거의 독립

서로 다른 좌표 $u_i$와 $u_j$ ($i \neq j$):

$$\text{Cov}(u_i, u_j) = -\frac{1}{d(d-1)} \approx 0 \quad (d \text{ 클 때})$$

> **이것이 결정적**: 좌표가 거의 독립이면, **각 좌표를 독립적으로 양자화**해도 전체적으로 좋은 성능을 달성할 수 있습니다. 벡터 양자화의 성능을 스칼라 양자화로 달성!

## 6. 시각적 요약: 랜덤 회전의 효과

```
┌─────────────────────────────────────────────────────────┐
│                   랜덤 회전의 3중 효과                     │
│                                                          │
│  1. 분포 균등화                                           │
│     불규칙한 분포 → 예측 가능한 Beta/가우시안 분포          │
│     → 최적 양자화기를 미리 설계 가능                       │
│                                                          │
│  2. 좌표 간 독립화                                        │
│     상관된 좌표들 → 거의 독립인 좌표들                      │
│     → 스칼라 양자화로 벡터 양자화 근사                     │
│                                                          │
│  3. 정규화 불필요                                         │
│     분포가 데이터와 무관 → min/max/scale 저장 불필요       │
│     → 양자화 상수 오버헤드 = 0                            │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 7. 이 강의에서 기억할 것

1. **고차원의 직관**: 대부분의 부피가 껍질에, 랜덤 벡터는 거의 직교, 좌표값은 0 근처 집중
2. **JL Lemma**: 랜덤 투영은 쌍별 거리를 보존 — $k = O(\varepsilon^{-2} \log n)$
3. **측도의 집중**: 고차원에서 랜덤 투영의 노름이 평균에 지수적으로 빠르게 집중
4. **랜덤 회전의 3중 효과**: 분포 균등화 + 좌표 독립화 + 정규화 불필요
5. **Data-Oblivious**: 회전 후 분포가 데이터와 무관 → 캘리브레이션 없이 양자화 가능
